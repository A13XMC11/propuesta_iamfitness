# Propuesta IAM Fitness

Landing/presentación estática para la propuesta creativa de IAM Fitness.

## Estructura

- `index.html`: archivo principal listo para publicar en Vercel.
- `iam_fitness_presentation_v3.html`: versión fuente original de la presentación.
- `vercel.json`: configuración mínima para despliegue estático.

## Desarrollo local

Puedes abrir `index.html` directamente en el navegador o levantar un servidor local:

```bash
npm install
npm run dev
```

## Despliegue en Vercel

1. Sube este repositorio a GitHub.
2. En Vercel, crea un nuevo proyecto desde el repositorio.
3. Usa la configuración por defecto. No requiere build command.
4. Publica el proyecto.
